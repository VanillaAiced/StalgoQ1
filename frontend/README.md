# Midnight Sun Café

A modern, responsive website for Midnight Sun Café built with React. Features a clean design with smooth navigation, interactive menu carousel, and gallery showcase.

## Setup

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

The site will open at [http://localhost:3000](http://localhost:3000).

## Build

Create a production build:

```bash
npm run build
```

## Tech Stack

- React 18.3
- React Router 6.22
- Tailwind CSS 3.4
- Lucide React icons

## Project Structure

```
src/
├── components/       # React components (Header, Hero, About, etc.)
├── data/            # Site content and configuration
└── images/          # Image assets
```

## Features

- Responsive navbar with scroll effects
- Menu page with image carousel
- Gallery section with alternating layouts
- Contact information with social links
- Mobile-friendly design
